inlinestatic

Constructor, requires keys and values of the function to interpolate.

For simplicity and compatibility with the predefined ControllerValue classes the function domain must be [0,1]. However, you can use the frequency parameter to rescale the domain to a different range.

Parameters
keysthe x-values of the function sampling points. Value range is [0,1]. Must include at least the keys 0 and 1.
valuesthe function values f(x) of the function. order must match keys
frequencythe speed of the evaluation in cycles per second
deltaInputIf true, signifies that the input will be a delta value such that the function should add it to an internal counter before calculating the output.
Note
there must be the same amount of keys and values
